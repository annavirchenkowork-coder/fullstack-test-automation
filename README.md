# QA_Github Project

## Overview
This is a scalable QA Automation Framework for UI, API, and database testing.

## Tech Stack
- Selenium WebDriver
- Java, TestNG, Maven
- Playwright
- REST Assured / Postman
- Jenkins & GitHub Actions
- SQL/MySQL validation
- ExtentReports / HTML Reports

## How to Run
1. Clone the repo
2. Install dependencies with Maven
3. Run test classes via TestNG
4. Reports will generate in `/reports` folder

## Additional Features
- Cross-browser automation with Playwright
- REST Assured API testing
- Database validation with MySQL
- CI/CD with GitHub Actions
